# SQL Customer Churn Analysis

## Objective
  Analyze customer data to understand why customers leave and identify ways to improve retention.
## Tool Used
 **MY SQL**
## Dataset
  - 5,000+ customer records
  - Includes: demographics, orders, complaints, transactions, and satisfaction scores
## Tasks Performed
- Handled missing values and inconsistent labels  
- Created derived customer indicators  
- Performed grouped analysis using aggregations, joins, and subqueries
## Key Insights
  - Customers who raised complaints are more likely to churn
  - Customers far from warehouses tend to leave more
  - Higher churn observed in Tier-3 regions for some products
  - Debit card users show better retention
## Recommendations
  - Improve customer support and complaint handling
  - Optimize delivery for far locations
  - Focus on Tier-3 market issues
  - Continue strong payment experience for debit card users
